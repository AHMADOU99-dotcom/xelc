vous souhaitez apprendre le HTML et CSS? Vous etes sur le bon site:) Ici, nous allons voir les bases de ces deux langages, puis apprendre plus en profondeur les spécificités de chacun. Les cours se découpent selon un ordre logique qui va vous permetre d'apréhendre ces deux langages le plus serainement possible:
  1. Un introduction du fonctionnement du HTML, et du CSS, pour bien comprendre à quoi sert l'un et l'autre.
  2. Les bases du fonctionnement du HTML, pour maitrise sa syntaxe.
  3. Les bases du fonctionnement du CSS, pour maitrise sa syntaxe qui est different du HTML
  4. La mise en forme du texte, qui nous permet de faire le lien entre le HTML et CSS.
  5. Le positionnement, qui nous permet de structurer un page web.
Arrivé à ce stade, vous aurez déjà compris comment fonctionnnent ces deux langages, et vous pourrez volez de vos propre ailes. On vous propose tout de meme d'autre lecons pour se perfectionner dans les domaines suivants:
  1. La gestion des fonds;
  2. Les balises video et audio,
  3. Les tableaux,
  4. Les formulaires,
  5. Comment gérer les responsive
Enfin, pour cloturer ces cours, vous avez deux derniéres thématiques:
  1. Le programme avancé de CSS pour devenir un cador,
  2. La mise en production de son code serveur.
