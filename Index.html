Voici la liste des cours que nous disposons:

HTml

Le langage utilisé pour structurer une page web. On peut par exemple organiger le contenu en un ensemble de paragraphe, une liste d'elements, utiliser des images ou des tableaux de données.

Java

Il alimente les applications, les systemes d'exploitation smartphone, les logiciels d'entreprise et de nombreux programmes connu. Meme s'il y plus de 20ans, Java est actuellement le lagage de programmation le plus populaire pour les developpeurs d'application.

JAVAscript

Javascript est un langage de programmation qui permet de créer du contenu mis a jour de facon dynamique, de controle le contenu multimédia d'animer des images, et tout ce a quoi on peut penser. Bon, peut etre pas tout, mais vous pouvez faire bien des choses avec quelques lignes de Javascript.

PHP

PHP est un langage de programmation libre, principalement utilissé pour produire des pages web dynamiques via un serveur web, mais pouvant également fonctionner comme n'importe quel langage interprété de facon locale. PHP est un langage impératif orienté objet

UML

UML st un langage de modélisation standarisé qui peut etre utilisé pour différents evironnements de programmation et processus de developpement. il est concu pour que la majorité des developpeurs de logiciels le comprennent et soient en mesure de l'appliquer à leur travail

CSS

Le css correspond à un langage informatique permettant de mettre en forme des pages web (HTML ou XML)
